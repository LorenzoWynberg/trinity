# Activity Logs

Development activity logs organized by project.

## Structure

```
logs/activity/
├── README.md       # This file
├── trinity/        # Trinity CLI tool activity logs
│   ├── README.md
│   ├── YYYY-MM-DD.md
│   └── archive/
└── ralph/          # Ralph development loop activity logs
    ├── README.md
    ├── YYYY-MM-DD.md
    └── archive/
```

## Projects

- **trinity/** - Activity logs for Trinity CLI tool development
- **ralph/** - Activity logs for Ralph (autonomous development loop)

## Dashboard

The Ralph dashboard displays activity logs from both projects with a toggle to switch between them. Trinity logs are shown by default.
