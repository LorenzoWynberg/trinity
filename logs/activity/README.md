# Activity Logs

Development activity logs organized by project.

## Structure

```
logs/activity/
├── README.md       # This file
├── trinity/        # Trinity CLI development (written by Ralph)
│   ├── README.md
│   ├── YYYY-MM-DD.md
│   └── archive/
└── ralph/          # Ralph development (written by humans)
    ├── README.md
    └── YYYY-MM-DD.md
```

## Projects

- **trinity/** - Activity logs for Trinity CLI tool development, written by Ralph
- **ralph/** - Activity logs for Ralph's own development, written by humans

## Dashboard

The Ralph dashboard displays activity logs from both projects with a toggle to switch between them. Trinity logs are shown by default.
